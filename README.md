## Instalasi

Setelah melakukan clone repo ini, buat database kosong dengan nama: 
```bash
wisataBMS
```
Lalu konfigurasi file .env dan lakukan migrate serta seeding:
```bash
php artisan migrate
php artisan db:seed
```
